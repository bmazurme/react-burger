### Installation
```bash

# clone
$ git clone https://github.com/bmazurme/react-burger

# install dependencies
$ npm install

# build project
$ npm run build

# run dev mode
$ npm run dev

# run prod mode
$ npm run start

# run eslint
$ npm run eslint

# run stylelint
$ npm run stlint

# clear
$ npm run clear
```
